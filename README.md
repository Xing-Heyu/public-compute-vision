## 🧭 一、项目定位：一个已经跑通、可自部署的技术工具包

**2026年2月，我们完成了以下核心验证：**

✅ **跨平台闲置检测库 `idle-sense`**  
Windows/macOS 可稳定运行，准确识别“真·闲置”状态。**现已发布 v0.1.0，可独立安装使用。**

网页提交 → 调度中心匹配 → 节点执行 → 结果返回。**这条路，已经走通。**

✅ **调度器可自部署**  
任何人都可以：
- 下载本仓库代码
- 在自己机器上 `python scheduler/simple_server.py`
- **5分钟内拥有完全属于自己的调度中心**

✅ **文件夹隔离与自动清理**  
用户文件与任务缓存严格分离，任务结束后自动销毁临时数据。

✅ **核心代码完全开源，MIT协议**  
**这不是实验室demo，这是已公开、可复现、可立即使用的开源原型。**

## 🚀 四、如何立即开始

### ✅ 5分钟自建调度中心

```bash
# 1. 克隆仓库
git clone https://github.com/你的用户名/idle-accelerator
cd idle-accelerator

# 2. 启动调度器
python scheduler/simple_server.py

# 3. 启动节点客户端（Windows/macOS）
python node/windows_client.py   # Windows
# 或
python node/simple_client.py    # macOS/Linux

# 4. 启动网页界面
streamlit run web_interface.py
```

**打开 `http://localhost:8501`，即可提交任务。**
